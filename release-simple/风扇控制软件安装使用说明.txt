#软件安装
	run安装|卸载：
		1. chmod +x FanControlInstaller_ctyun.run
		2. ./FanControlInstaller_ctyun.run install
		3. 卸载： ./FanControlInstaller_ctyun.run uninstall
		       注意：麒麟系统卸载时，删除文件有权限问题，需额外运行命令：rm /usr/local/bin/AutoFanCtrl /usr/local/bin/ManFanCtrl /lib/systemd/system/autofanctrl.service
		       
	源码编译安装：
		1. make
		2. make install
		3. systemctl daemon-reload
		4. systemctl enable autofanctrl.service
		5. systemctl start autofanctrl.service
		
	源码编译卸载：
		1. systemctl stop autofanctrl.service
		2. systemctl disable autofanctrl.service
		3. systemctl daemon-reload
		4. make clean
		5. make uninstall
		
#软件使用
	自动模式：
		安装完毕后，它已被设置为开机自动启动程序并根据温度调整风扇，无需其他操作
		如果AI_CARD1和AI_CARD2不插卡，PWM恒定20,插卡PWM恒定50
	手动模式：
		帮助手册：ManFanCtrl -h
		查看温度： ManFanCtrl -t <device_name> ，受支持的设备名： cpu, mainboard
		查看功率： ManFanCtrl -p <device_name> ，受支持的设备名： cpu
		设置风扇PWM： ManFanCtrl -s <device_name> <PWM> ，受支持的设备名： cpu, sysFan, AI_CARD1, AI_CARD2，命令示例： ManFanCtrl -s cpu 10
		切换为自动模式： ManFanCtrl -a
